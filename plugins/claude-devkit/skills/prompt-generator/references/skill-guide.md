# 스킬 생성 프롬프트 가이드

## 프롬프트 구조

```markdown
# [스킬명] 스킬 생성 요청

## 목적
[스킬이 해결하려는 문제/제공하려는 기능]

## 트리거 조건
[어떤 상황에서 이 스킬이 활성화되어야 하는지]
- 키워드: ...
- 맥락: ...

## 핵심 기능
1. [기능 1]
2. [기능 2]
...

## 입력/출력
- 입력: [사용자가 제공하는 정보]
- 출력: [스킬이 생성하는 결과물]

## 참고 자료
- [기존 유사 스킬/문서가 있다면 명시]

## 제약사항
- [기술적 제한, 규칙 등]
```

## 좋은 프롬프트 예시

```markdown
# PDF 처리 스킬 생성 요청

## 목적
PDF 파일의 텍스트 추출, 페이지 회전, 병합 기능 제공

## 트리거 조건
- 키워드: PDF, 페이지 회전, PDF 병합, 텍스트 추출
- 맥락: 사용자가 PDF 파일 경로를 언급하거나 PDF 작업 요청 시

## 핵심 기능
1. 텍스트 추출 (pdfplumber 사용)
2. 페이지 회전 (90/180/270도)
3. 여러 PDF 병합

## 입력/출력
- 입력: PDF 파일 경로, 작업 유형
- 출력: 처리된 PDF 또는 추출된 텍스트

## 제약사항
- Python 환경 필요
- 암호화된 PDF는 비밀번호 필요
```

## 핵심 포인트

1. **description 필드 중요**: 스킬 트리거의 핵심. 사용 시나리오를 구체적으로 기술
2. **점진적 공개**: SKILL.md는 핵심만, 상세는 references/에 분리
3. **재사용 자원 식별**: scripts/, references/, assets/ 중 필요한 것 파악
